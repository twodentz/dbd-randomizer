# Dead by Daylight Randomiser

A Dead by Daylight themed character and perk randomiser built with vanilla HTML, CSS, and JavaScript.

This tool allows players to randomly generate Survivors or Killers along with unique perk loadouts, with quality of life features such as rerolling individual perks, viewing perk and character details, and toggling Legendary characters on or off.

---

## Development

Active development happens on the `dev` branch.

The `main` branch is kept stable and represents the latest “ready” version of the project.

---

## Features

### Randomisation
- Random Survivor or Killer selection
- Random perk loadouts (no duplicates)
- Reroll:
  - Entire loadouts
  - Individual perks
  - Character only

### Smart Logic
- Prevents rolling the same character twice in a row
- Guarantees unique perks per loadout
- Optional toggle to include or exclude **Legendary characters**

### Modals & Info
- Clickable perks with detailed descriptions
- Clickable characters with:
  - Portrait
  - Biography
  - Chapter information

### Design
- Fully custom Dead by Daylight inspired UI
- Subtle animations for characters and perks
- Responsive layout for smaller screens
- Tooltips for UI clarity
